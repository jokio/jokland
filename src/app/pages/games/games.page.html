Web3 Games

We believe online games should work the decentralized way as a protocol. Think about Poker protocol, or TikTakToe protocol, like email protocols (SMTP, POP3). This means everyone can write their client and use the same protocol.

Our vision is to store all assets on IPFS (like this website). To keep all NFTs in Blockchain. The only missing piece is a real-time communication mechanism for online games.

We plan to solve this problem and provide a decentralized real-time communication solution. Everyone will be able to use our js SDK and create web3 online games.

We think games are minimal building blocks, which should be free for everyone to play and lots of abstractions will be built on top of it in a near future.
